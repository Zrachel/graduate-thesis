zjuthesis
=========

Latex template for zju thesis

具体使用方法请阅读 [README.pdf](https://github.com/ibillxia/zjuthesis/blob/master/README.pdf) 文档。


### 补充说明

在使用之前需要对 ctex-xecjk-winfonts.def 文件进行修改，这里你只需要将本项目中的 ctex-xecjk-winfonts.def 文件拷贝到以下目录（覆盖之前请自行备份）：

`%CTEXDIR%/MiKTeX/tex/latex/ctex/fontset`

其中%CTEXDIR%是CTeX的安装目录。


此外，推荐Windows用户使用号称所见及所得的 [TexMakers](http://www.xm1math.net/texmaker/) 作为 tex 文件的编辑器，确实非常好用。
